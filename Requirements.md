

### JSON Tools Web App Prompt

Create a modern, professional, fully client-side JSON Tools web application using HTML, CSS, and JavaScript.

Requirements:

#### Core Features

1. JSON Formatter / Beautifier

   * Format JSON with proper indentation.
   * Support 2-space and 4-space indentation.
   * Preserve Unicode characters.
   * Display formatted output instantly.

2. JSON Minifier

   * Remove unnecessary whitespace.
   * Generate compact JSON output.

3. JSON Validator

   * Validate JSON syntax.
   * Display clear error messages.
   * Show line and column number for errors.

4. JSON Parser

   * Parse JSON safely.
   * Display parsed structure in a readable tree view.
   * Expand/collapse nodes.

5. JSON Fixer

   * Detect common JSON mistakes:

     * Missing commas
     * Trailing commas
     * Single quotes
     * Unquoted property names
   * Suggest fixes.
   * One-click auto-fix when possible.

6. JSON Tree Viewer

   * Interactive hierarchical tree.
   * Expand all / Collapse all.
   * Search within tree.

7. JSON Comparison Tool

   * Compare two JSON documents.
   * Highlight differences.
   * Side-by-side view.

8. JSON Search

   * Search keys and values.
   * Highlight matching results.

9. JSON to Other Formats

   * JSON → CSV
   * JSON → XML
   * JSON → YAML

10. File Support

    * Upload .json files.
    * Drag-and-drop support.
    * Download formatted output.
    * Copy to clipboard.

#### Offline Requirements

* Must work completely offline.
* No backend.
* No API calls.
* No external services.
* All processing happens in the browser.
* Store user preferences in LocalStorage.

#### User Interface

* Modern responsive design.
* Desktop and mobile friendly.
* Light mode and dark mode.
* Split-pane editor layout.
* Syntax highlighting.
* Line numbers.
* Resizable panels.
* Keyboard shortcuts:

  * Ctrl+Enter → Format
  * Ctrl+Shift+M → Minify
  * Ctrl+F → Search
  * Ctrl+C → Copy output

#### Performance

* Handle JSON files up to 20MB.
* Fast rendering.
* Debounced validation.
* Efficient tree rendering for large datasets.

#### Accessibility

* WCAG-compliant colors.
* Keyboard navigation.
* Screen-reader support.
* Proper ARIA labels.

#### Technical Requirements

* Pure HTML, CSS, and Vanilla JavaScript.
* No backend required.
* Modular code structure.
* Clean comments and documentation.
* Error handling throughout.
* Use modern ES6+ syntax.
* Produce a complete ready-to-run project.

#### Output Structure

Generate:

* index.html
* styles.css
* script.js

Include:

* Complete source code.
* Responsive styling.
* All functionality implemented.
* Sample JSON examples.
* Usage instructions.

The final application should look comparable to professional online JSON tools while remaining fully offline and self-contained in a single local website.
